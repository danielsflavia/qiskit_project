# Quantum Random Number Generator (Qiskit)

## Requirements

### **Python Version**

You **must** use:

```bash
Python 3.12
```

### Virtual Environment Setup

```bash
python3.12 -m venv .venv

source .venv/bin/activate

pip install --upgrade pip

pip install qiskit

pip install qiskit-aer

```
